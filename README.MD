## 项目介绍
技术栈：  
>`Spring Boot3.0`、`Spring Security`、`MyBatis`、`MyBatis Plus`、`MySQL`、`Redis`、`Lombok`

技术架构：  
> 项目采用前后端分离模式开发，基于 Spring Boot 框架搭建 RESTful 风格 API，规范使用状态码
与JSON数据格式  

安全认证：
>基于 JWT 实现 Token 认证机制，采用 Spring Security 配置白名单与鉴权逻辑，通过自定义 User
DetailsService 加载用户权限，结合 OncePerRequestFilter 实现 Token 解析与状态验证，保障接口安全性

系统防护：
>通过拦截每次请求实现 IP 限流功能，结合 Redis 记录单位时间请求次数，有效拦截恶意请求，
减少服务器消耗

异常处理：
>使用 @RestControllerAdvice 实现全局异常处理，统一封装 ResultVO 响应对象，标准化错误码
体系，了解每次异常来源及原因

日志优化：
>通过 AOP 切面编程实现日志打印功能，使用自定义 @SysLog 注解标记业务方法及异常处理方
法，采集请求参数、请求路径、请求 IP 、请求方法等关键指标，提升问题排查效率

事务控制：
>使用 @Transactional 注解保证增删改操作时时的原子性操作，异常回滚率100%，保护数据库数
据完好
